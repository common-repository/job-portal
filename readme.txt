==Description==
Job portal allows web admin to publish jobs. Job Portal contain list template to list all job posted. A side bar widget is also include in job portal which can also be integrated with web site front page.
<b>Job-portal Free Features </b>
<ul>
	<li>Awesome job listing page to show all jobs posted by admin of website</li>
	<li>Simple and elegant single page design which can be integrated with any website</li>
	<li>Some required property in job post address like (job location, job nature(full time, part time etc), Experience Req, Job category etc.)</li>
	<li>Awesome plugin widget that also be integrated as required in required area and space.</li>
</ul>


== Installation ==

This section describes how to install the plugin and get it working.

1.Upload and install job-portal plugin .zip file from admin dash board.
<br>
2.After sucessfully installing jop-portal use can now see a new nav menu with name jobs on admin dash board.
<br>
3.To post a new job, fill the job form with required fields and publish.
<br>
4.After publishing first job creat a new page name as Avalible job or career etc  but don,t forget to select Job listing page form template and save pblish. 
<br>
5.To set-up job-portal widget inclcude widget from appereance, here you find two text field first  one Widget Text label and second count( how many jobs appear in widget area).
<br>
6.Done, you finally installed the job-portal plugin successfully. Enjoy.
